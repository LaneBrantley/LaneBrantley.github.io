Project Name: Andrew's Professional Opinions

Project Overview:

This application will be used to display Andrew Cipolla's Professional opinions. Users will be anybody looking advice in mathematics or life advice. There will be dropdowns that the user can click on to get a response from the website, and if they need to perform mathematical experssions, there will be boxes for that.

Site Map:

Page Design:

Home Page: Will host most of what the user will see during their interaction. Will have some data entry so it will require data validation via JS.

Terms of Service: Will host the TOS for the company.

About us: Will display the information about the company/client.

Support: Will have basic information about issues and links for more specific things.

Contact Support: Will be various ways to contact us for support.

Forums: Will be a space for users to post about issues they've had. No data validation as it is simply text, no need to validate.

Manual: Basic Information on how to use the website and what it's for.

Dynamic Functionality on the Website:

For the Home Page, there will need to be interactivity between the page and JavaScripts, both to validate the input and to provide feedback on the input. The forums will also need JavaScript to fill in the page based off of what the user entered.
